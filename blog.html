Difference between localStorage and sessionStorage

Data in LocalStorage does not expire but data in sessionStorage is cleared when the page session ends. localStorage capacity is 5MB/10MB but sessionStorage capacity is 5MB.localStorage is not session based, need to be delete by JS or Manually but sessionStorage is session based.

Difference between Global scope and Block scope

Variables declared Globally (outside any function) have Global Scope. Variables declared inside a { } block that is block Scope. Global scope's variables can be accessed from anywhere in a JavaScript program but Block Scope's variables can not accessed form outside the block
